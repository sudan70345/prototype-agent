# 供应商管理模块 — 页面功能原型设计描述

---

## 页面1: 供应商信息管理

> Pencil 页面名: P1_供应商信息管理

### 布局结构
- **整体布局**: 顶部通用栏目 + 左侧固定菜单栏 + 右侧数据内容区
- **固定元素**: 
  - 顶部栏目（Logo + 系统名称 / 用户名 + 退出）
  - 左侧菜单栏（一级"供应商管理"，子菜单含"供应商信息管理"；其余一级菜单占位）
- **滚动方式**: 右侧内容区纵向滚动，页面整体无横向滚动

### 模块清单（按优先级排列）

#### 🔴 重要模块

**1. 搜索筛选栏**
   - **类型**: form（水平排列）
   - **位置**: 表格上方
   - **内容**: 
     - 供应商名称（文本输入框）
     - 供应商分类（下拉选择，多选，数据来源字典）
     - 供应商等级（下拉单选：金牌/银牌/铜牌）
     - 状态（下拉单选：启用/停用）
   - **按钮**: 「搜索」「重置」「新建」
   - **设计要点**: 蓝绿/青色系主色调，Ant Design 风格

**2. 供应商数据表格**
   - **类型**: table
   - **列字段**（共 10 列）:
     | 列名 | 说明 | 备注 |
     |-----|------|------|
     | 供应商编码 | 文本 | — |
     | 供应商名称 | 文本 | — |
     | 联系人 | 文本 | — |
     | 联系电话 | 文本 | **脱敏展示**（如：138****5678） |
     | 供应分类 | 标签/枚举 | 多分类以标签形式展示（如：🏷️乳制品 🏷️肉类） |
     | 等级 | 标签 | 金牌/银牌/铜牌，用不同颜色区分 |
     | 状态 | 标签 | 启用/停用，不同颜色区分 |
     | 创建时间 | 文本 | 格式化时间 |
     | 修改时间 | 文本 | 格式化时间 |
   - **操作列**: 固定在最右侧，每行包含「编辑」「删除」两个文本链接
   - **设计要点**: 每行展示完整供应商信息，行内容宽度不超过容器

**3. 分页组件**
   - **类型**: pagination
   - **位置**: 表格右下方（与表格右边缘对齐）
   - **设计要点**: 标准 Ant Design 分页组件

#### 🟡 次要模块

**4. 顶部通用栏目**
   - 左侧: Logo + "xxx公司进销存管理系统"
   - 右侧: 当前登录用户名 + "退出"链接

**5. 左侧菜单栏**
   - 一级菜单「供应商管理」（展开状态）
   - 子菜单: 「供应商信息管理」（选中态）
   - 其他一级菜单占位留空

### 关键元素
- **搜索输入框**: 供应商名称（文本框）、供应商分类（多选下拉）、等级（下拉）、状态（下拉）
- **操作按钮组**: 搜索、重置、新建
- **表格操作列**: 编辑、删除（文本链接形式）
- **联系电话**: 脱敏展示（138****5678 格式）
- **供应分类标签**: 多分类以彩色标签形式展示

### 交互定义

| 触发元素 | 交互类型 | 目标/反馈 | 说明 |
|---------|---------|----------|------|
| 搜索按钮 | click | 刷新表格数据 | 按筛选条件查询 |
| 重置按钮 | click | 清空条件，刷新表格 | 恢复默认列表 |
| 新建按钮 | click | 打开新增供应商大弹窗 | 表单弹窗，分两区 |
| 编辑链接（操作列） | click | 打开编辑供应商大弹窗 | 预填供应商数据 |
| 删除链接（操作列） | click | 打开删除确认小弹窗 | "是否确认删除该供应商？" |
| 弹窗确认按钮 | click | 执行操作并关闭弹窗 | 成功后刷新表格 |
| 弹窗取消/放弃按钮 | click | 关闭弹窗，不执行 | 取消操作 |

### 弹窗详细设计

#### 新增/编辑大弹窗
> ⚠️ 此弹窗在阶段3将作为独立的 Pencil 页面存在于 .pen 文件内，命名为 P1D1_新建编辑供应商

- **Pencil 页面名**: P1D1_新建编辑供应商
- **触发方式**: 点击「新建」按钮或操作列「编辑」链接
- **弹窗类型**: 表单弹窗
- **弹窗标题**: 新建供应商 / 编辑供应商（根据操作类型动态切换）
- **画布尺寸**: 宽 800px × 高 520px
- **表单布局**: 分两区

  **基本信息区**（前 6 个字段）:
  1. 供应商编码（文本输入框，必填）
  2. 供应商名称（文本输入框，必填）
  3. 统一社会信用代码（文本输入框，必填）
  4. 联系人姓名（文本输入框，必填）
  5. 联系电话（文本输入框，必填）
  6. 联系地址（文本输入框，必填）

  **经营信息区**（后 7 个字段）:
  7. 供应分类（多选下拉选择框，来自字典数据）
     - 可选分类: 乳制品、肉类、水产、蔬菜、水果、粮油、饮料、调料、冻品、零食、其他
  8. 供应商等级（下拉单选：金牌 / 银牌 / 铜牌）
  9. 经营范围（多行文本框，可选）
  10. 开户银行（文本输入框，可选）
  11. 银行账号（文本输入框，可选）
  12. 状态（下拉单选：启用 / 停用，默认"启用"）

- **底部按钮**: 「保存」（主按钮，右）/ 「取消」（次按钮，右）

#### 删除确认弹窗
> ⚠️ 此弹窗在阶段3将作为独立的 Pencil 页面存在于 .pen 文件内，命名为 P1D2_删除确认

- **Pencil 页面名**: P1D2_删除确认
- **触发方式**: 点击操作列「删除」链接
- **弹窗类型**: 确认弹窗
- **画布尺寸**: 400×180
- **标题**: 确认删除
- **内容区**: "是否确认删除该供应商？删除后数据无法恢复"
- **按钮**: 「确认」（主按钮，右）/ 「取消」（次按钮，右）
