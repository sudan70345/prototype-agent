# 集采商城_V1 — 需求分析文档

## 1. 产品背景

- **产品名称**: 集采商城
- **产品定位**: 企业集中采购管理平台，用于内部物资采购、商品浏览、购物车管理
- **目标用户**: 企业内部采购人员（无内外部用户之分，统一为已认证内部账号）
- **适配平台**: 桌面 Web（浏览器，最小适配宽度 1280px）

## 2. 页面清单

| 编号 | 页面名称 | 功能定位 | 优先级 | 访问用户 |
|------|---------|---------|--------|---------|
| page_001 | 登录页 | 身份验证入口 | P0 | 未登录用户 |
| page_002 | 密码找回页 | 密码找回流程 | P1 | 未登录用户 |
| page_003 | 商品物资展示页 | 商品浏览、搜索、分类筛选、加购 | P0 | 已登录用户 |
| page_004 | 购物车页面 | 购物车管理、批量操作、结算入口 | P0 | 已登录用户 |
| page_005 | 结算页（占位） | 结算流程入口，本期不做具体功能 | P2 | 已登录用户 |

## 3. 页面调用路径

```
未登录状态:
  ┌─────────────────────────────────────────────────────────┐
  │ page_001 登录页                                         │
  │   │                                                     │
  │   ├──[登录成功]──→ page_003 商品物资展示页              │
  │   │                                                     │
  │   └──[点击"忘记密码"]──→ page_002 密码找回页            │
  │                                                         │
page_002 密码找回页                                          │
  │   │                                                     │
  │   └──[找回成功]──→ page_001 登录页                      │
  └─────────────────────────────────────────────────────────┘

已登录状态:
  ┌─────────────────────────────────────────────────────────┐
  │ page_003 商品物资展示页                                 │
  │   │                                                     │
  │   ├──[点击商品卡片]──→ 商品详情弹窗（模态框，不跳转）    │
  │   │                                                     │
  │   ├──[点击"加入购物车"]──→ 购物车 +1 toast 提示         │
  │   │                                                     │
  │   └──[点击购物车入口]──→ page_004 购物车页面            │
  │                                                         │
page_004 购物车页面                                          │
  │   │                                                     │
  │   └──[点击"去结算"]──→ page_005 结算页（占位）          │
  └─────────────────────────────────────────────────────────┘
```

## 4. 交互关系定义

| 触发元素 | 交互类型 | 目标 | 说明 |
|---------|---------|------|------|
| 登录按钮 | click | — | 校验账号密码，成功跳转商品页 |
| 忘记密码链接 | click | page_002 | 跳转密码找回页 |
| 手机号输入框 | input | — | 输入11位手机号 |
| 获取验证码按钮 | click | — | 发送短信验证码 |
| 验证码输入框 | input | — | 输入6位验证码 |
| 新密码/确认密码输入框 | input | — | 密码强度校验，两次一致才允许提交 |
| 提交密码修改 | click | — | 成功后跳转登录页 |
| 分类列表项 | click | — | 切换分类，过滤下方商品 |
| 搜索框 | input/keyup | — | 按名称实时搜索/筛选 |
| 商品卡片 | click | 弹窗 | 点击打开商品详情弹窗 |
| 商品卡片加购按钮 | click | — | 弹窗提示加购成功，购物车角标+1 |
| 商品数量输入框 | input | — | 设置购买数量 |
| 商品详情弹窗关闭 | click | — | 关闭弹窗返回商品页 |
| 购物车复选框 | click | — | 选中/取消选中单个商品 |
| 全选复选框 | click | — | 全选/取消全选 |
| 购物车数量增减 | click | — | 调整该商品数量 |
| 删除单个商品 | click | — | 移除此商品 |
| 批量删除按钮 | click | — | 删除所有已选商品 |
| 去结算按钮 | click | page_005 | 进入结算页（占位） |
