## page_002 — 密码找回页

### 功能清单
- F-001 步骤指示器（3 步：手机号→验证码→新密码）
- F-002 手机号输入框
- F-003 获取验证码按钮 + 倒计时
- F-004 验证码输入框
- F-005 新密码输入框
- F-006 确认密码输入框
- F-007 提交按钮
- F-008 返回登录链接

### 布局结构
- 布局类型: 居中卡片布局
- 容器宽度: 420px，居中于视口
- 区块自上而下:
  1. step_indicator — 步骤指示器
  2. dynamic_form — 动态表单（随步骤变化）
  3. back_link — 返回登录链接

### 模块设计要求

```
┌─────────────────────────────────────────────┐
│ 模块: step_indicator                          │
│ 类型: stepper (3 步)                          │
│ 宽度: 100% / 高度: 32px                        │
│ 步骤:                                          │
│   ① 输入手机号 → ② 验证手机 → ③ 设置新密码    │
│ 当前步骤: 高亮主色 #1D4ED8                    │
│ 已完成步骤: 绿色 #059669                      │
│ 未完成步骤: 灰色 #9CA3AF                      │
└─────────────────────────────────────────────┘

┌─────────────────────────────────────────────┐
│ 模块: dynamic_form — 步骤1: 手机号输入         │
│ 内容:                                          │
│   - 手机号输入框 — placeholder: "请输入手机号"  │
│     高度: 44px / 圆角: 8px                    │
│   - 获取验证码按钮 — 主色                      │
│     宽度: 100% / 高度: 44px / 圆角: 8px       │
│     点击后变为倒计时 60s                       │
│   点击"下一步" → 进入步骤2                     │
└─────────────────────────────────────────────┘

┌─────────────────────────────────────────────┐
│ 模块: dynamic_form — 步骤2: 验证码输入         │
│ 内容:                                          │
│   - 验证码输入框 — 6 位数字                    │
│     高度: 44px / 圆角: 8px                    │
│     分格输入样式（6 个格子）                   │
│   - 重新发送链接 — 字号 14px                   │
│   验证码正确 → 进入步骤3                       │
└─────────────────────────────────────────────┘

┌─────────────────────────────────────────────┐
│ 模块: dynamic_form — 步骤3: 设置新密码         │
│ 内容:                                          │
│   - 新密码输入框 — placeholder: "请输入新密码"  │
│     高度: 44px / 圆角: 8px                    │
│     显示密码强度指示条                           │
│   - 确认密码输入框 — placeholder: "请再次输入"  │
│     高度: 44px / 圆角: 8px                    │
│   - 提交按钮 — 主色，密码一致且满足强度才可点击  │
│     宽度: 100% / 高度: 44px / 圆角: 8px       │
│   提交成功 → 跳转 page_001 登录页             │
└─────────────────────────────────────────────┘
```

### 功能交互
- 步骤 1 手机号输入 → 点击"获取验证码" → 60s 倒计时
- 步骤 2 验证码输入 → 校验 6 位数字 → 正确进入步骤 3
- 步骤 3 双密码输入 → 密码一致且长度≥8 → 提交成功跳转登录页
